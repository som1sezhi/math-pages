welcome to the (in)finite

The idea that "some infinities are bigger than other infinities" sounded pretty wild the first time I heard of it. Infinity is a concept, not a number, right? So how can you compare them like numbers and proclaim one infinity as bigger than the other?

Well, from my current understanding, infinity is just a concept, but that doesn't stop transfinite numbers from existing:

Definition 1-1: A transfinite number is a number greater than any finite number.

To see how transfinite numbers can come about, let's consider this set $A$:

beautiful

This lovely set $A$ here consists of four apples. Four is the number of elements in this set: its cardinality.

Definition 1-2: For a set $S$, its cardinality $|S|$ is the number of elements in $S$.

In the case of our previous set of apples, $|A|=4$.

Definition 1-3: Cardinal numbers (cardinals for short) are numbers that can be used to measure the cardinality of a set.

For example, any natural number (0 shall be considered a natural number, as is standard in set theory) can be used as a cardinal number; we can imagine a set quite easily having 0, 2, or a billion elements.

But what's the cardinality of $\mathbb N$, the set of all natural numbers?

$$\mathbb N=\{0,1,2,3,4,\ldots\}$$

"wow, that's a long 'un"

$\mathbb N$ is an infinite set, so we can't use any of the natural numbers to describe its cardinality. But $\mathbb N$ is a set, so it has a cardinality. Some cardinal number greater than any natural number...

$\aleph_0$ (aleph-zero, aleph-naught, or aleph-null) is the name we give to the cardinality of $\mathbb N$. Since $\aleph_0$ describes a cardinality, it is, in fact, a cardinal number, although it looks kinda funky compared to the numbers we're used to.

And so we reach our first transfinite number: the first, smallest infinity. But as you may have heard, infinity isn't as simple as just $\aleph_0$. We'll get to the bigger infinities when we get there, but in the meantime, we can categorize sets as follows:

Definition 1-4: A set $S$ is:

  • countable if $|S|\leq\aleph_0$
  • countably infinite if $|S|=\aleph_0$
  • uncountable or uncountably infinite if $|S|\gt\aleph_0$

A countable set is called "countable" since we can number off all its elements with the natural numbers, i.e. we can count their elements. Even in the cases where the set is countably infinite, you can still count off all its elements, although it'll take literally forever and you'll have to use all the natural numbers along the way.

But for uncountable sets? Note that uncountable sets have a larger cardinality than $\mathbb N$, meaning that even though the natural numbers are infinite, there are literally not enough naturals for you to number off all the elements of an uncountable set. We'll see in a later section how the hell a set can be like that.

Infinite sets are just some of those things in math where you'll need to throw standard intuition out the window to understand it. Still though, trying to wrap your head around one of these things can be quite fun!
